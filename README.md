Simulación de un juego de Pokémon por consola desarrollado en Java. El juego permite capturar Pokémon, gestionar el equipo, retar líderes de gimnasio y desafiar al Alto Mando, con persistencia de datos mediante archivos.

Taller/
├── Taller02_PO.java      # Clase principal y menú inicial
├── Partida.java          # Controlador del flujo del juego
├── Jugador.java          # Gestión del equipo y acciones del jugador
├── Pokemon.java          # Atributos y estadísticas de cada Pokémon
├── Gimnasio.java         # Lógica de batalla y desafío a líderes
├── AltoMando.java        # Secuencia de batallas consecutivas
├── lectorArchivos.java   # Lectura y escritura de archivos .txt
├── TablaTipos.java       # Matriz de efectividad de tipos (18x18)
├── Pokedex.txt
├── Habitats.txt
├── Gimnasios.txt
├── AltoMando.txt
└── Registros.txt
